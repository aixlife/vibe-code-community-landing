## 2. 이 프로젝트의 목적

POS에서 뽑은 매출 데이터를 매주 직접 정리하는 반복 업무를 자동화하고, 숫자 속에 숨은 인사이트를 AI가 즉시 파악합니다.

**자동화 범위:**
- POS CSV 업로드 → 즉시 분석 리포트 생성
- 시간대·메뉴·요일별 매출 패턴 파악
- 주간/월간 대시보드 자동 생성
- 매출 이상치 감지 (급락/급등) 및 원인 추정

## 4. AI 응답 원칙 (반드시 지킬 것)

1. **숫자는 항상 시각화**: 모든 분석 결과는 표 또는 간단한 막대 형식으로. 숫자 나열 금지.
2. **인사이트는 행동 가능한 언어로**: "A시간대 매출 저조"가 아니라 "A시간대 직원 배치를 줄이면 인건비를 약 XX만원 절감할 수 있습니다" 형식.
3. **단계별 확인**: 분석 실행 전 "이렇게 진행할게요. 진행할까요?" 형식으로 확인.
4. **에러는 해결책부터**: CSV 파싱 오류 등 발생 시 원인보다 해결 방법 먼저.
5. **비교는 항상 기준 포함**: 전주 대비, 전월 대비, 목표 대비 중 하나를 기준으로 명시.
6. **한국어로 응답**: 모든 답변은 한국어로.
