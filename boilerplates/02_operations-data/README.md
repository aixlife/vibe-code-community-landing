# 카테고리 2: 매장·서비스 운영 데이터 분석 보일러플레이트

**대상**: 매장 운영 대표님 (카페, 음식점, 펍, 소매, 뷰티샵 등)  
**핵심 가치**: POS 데이터를 매주 손으로 정리하는 대신, CSV 업로드 한 번으로 인사이트를 즉시 파악  
**수준**: 코딩 지식 불필요 (Antigravity + Claude로 진행)

---

## 이 보일러플레이트로 만들 수 있는 것

| 프롬프트 | 결과물 | 소요 시간 |
|---------|--------|----------|
| 01 CSV 즉시 분석 | 시간대·메뉴·요일 패턴 한눈에 파악 | 세션 내 30분 |
| 02 주간 리포트 | 전주 대비 매출 변화 + 인사이트 | 세션 내 30분 |
| 03 이상치 탐지 | 매출 급락/급등 알림 + 원인 추정 | 세션 내 20분 |
| 04 월간 대시보드 | 전략 인사이트 + 다음 달 액션플랜 | 세션 내 45분 |

---

## 사전 준비 사항

### 필수
- [ ] Antigravity 설치 및 Claude Pro 연결
- [ ] POS에서 CSV 내보내기 가능한지 확인
- [ ] 구글 계정 (결과 저장용 시트)

### CSV 파일 형식
POS마다 내보내기 형식이 다를 수 있습니다. `sample-data/pos-sample.csv`의 컬럼 형식을 참고해 맞춰주거나, 그대로 업로드하면 AI가 자동으로 파싱합니다.

**기본 권장 컬럼**: 날짜, 시간, 메뉴명, 카테고리, 수량, 단가, 합계, 결제수단, 테이블번호

### POS별 CSV 내보내기 방법
| POS | 경로 |
|-----|------|
| 포스뱅크 | 관리자 → 매출 → 기간별 내보내기 → CSV |
| 얼라이언스 | 백오피스 → 리포트 → 엑셀 다운로드 |
| 아임포스 | 설정 → 데이터 내보내기 → 매출 내역 |
| 직접 기록 | `sample-data/pos-sample.csv` 형식으로 작성 |

## 파일 구조

```
02_operations-data/
├── CLAUDE.md              ← Antigravity가 첫 번째로 읽는 컨텍스트
├── README.md              ← 이 파일
├── variables.md           ← 신청서 → 변수 매핑 가이드
├── prompts/
│   ├── 01-csv-upload-analysis.md  ← CSV 업로드 & 즉시 분석
│   ├── 02-weekly-report.md        ← 주간 리포트 자동 생성
│   ├── 03-anomaly-detection.md    ← 매출 이상치 탐지
│   └── 04-monthly-dashboard.md    ← 월간 대시보드
└── sample-data/
    └── pos-sample.csv     ← 샘플 데이터 (형식 참고용)
```

## 시작하는 방법

1. `CLAUDE.md`를 열고 `{}` 변수를 모두 채운다
2. POS에서 CSV를 내보낸다 (`sample-data/pos-sample.csv` 형식 참고)
3. Antigravity에서 이 폴더를 열고 `prompts/01-csv-upload-analysis.md`를 붙여넣는다
4. CSV 파일을 채팅창에 첨부하거나 경로를 알려준다
5. AI가 즉시 분석 결과를 제공한다
