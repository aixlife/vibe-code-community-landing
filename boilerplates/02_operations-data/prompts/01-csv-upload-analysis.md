# 프롬프트 01: CSV 업로드 & 즉시 분석

> 처음 시작하는 프롬프트입니다. POS CSV 파일을 첨부하면 즉시 분석 결과를 받습니다.
> 이 프롬프트 하나로 "우리 매장에 무슨 일이 일어나고 있는지" 전체 그림을 파악합니다.

---

## 사전 확인 (실행 전 체크)

- [ ] POS에서 CSV 파일을 내보냈다 (또는 sample-data/pos-sample.csv 준비)
- [ ] 분석할 기간을 정했다 (예: 지난 1주일, 지난달)

---

## 프롬프트 본문

```
당신은 {회사명}의 매출 데이터 분석을 담당하는 AI 에이전트입니다.

[매장 정보]
- 매장명: {회사명} ({매장수} 매장)
- 업종: {업종}
- POS 종류: {POS종류}
- 분석 기간: {분석기간}  <!-- 예: 2026-04-13 ~ 2026-04-19 (지난주) -->
- 분석 목적: {주요분석목적}

[첨부 파일]
아래에 CSV 파일을 첨부하거나 파일 경로를 알려주세요.
파일 경로: {CSV파일경로}  <!-- 또는 채팅창에 파일 첨부 -->

[분석 요청 사항]
CSV를 파싱한 뒤 다음 순서로 분석해주세요:

1. **데이터 요약** (파싱 결과 확인)
   - 총 거래 건수, 분석 기간, 컬럼 구성 확인
   - 이상한 데이터(결측값, 이상 금액 등) 발견 시 먼저 알려주세요

2. **전체 매출 요약**
   - 기간 총 매출액
   - 하루 평균 매출
   - 총 거래 건수, 평균 객단가

3. **시간대별 매출 분석**
   - 시간대를 2시간 단위로 묶어서 매출 합계 표로 정리
   - 피크타임 Top 3 시간대 강조
   - 비수기 시간대 파악

4. **요일별 매출 분석**
   - 요일별 매출 합계 및 평균 표
   - 가장 잘 팔리는 요일 / 가장 적은 요일

5. **메뉴(상품)별 매출 분석**
   - 매출 Top 10 메뉴 (금액 기준)
   - 수량 Top 10 메뉴 (수량 기준)
   - 매출 기여도 하위 메뉴 (정리 후보)

6. **결제 수단 분포**
   - 카드/현금/기타 비율 표

7. **AI 인사이트 3가지**
   - 지금 당장 활용할 수 있는 액션 위주로

모든 표는 마크다운 형식으로 깔끔하게 정리해주세요.
분석 시작 전에 "이렇게 진행할게요"로 계획을 먼저 말해주세요.
```

## 출력 형식 예시

### 전체 매출 요약 (2026.04.13 ~ 04.19)
| 지표 | 값 |
|------|---|
| 총 매출 | 18,450,000원 |
| 하루 평균 | 2,635,714원 |
| 총 거래 건수 | 523건 |
| 평균 객단가 | 35,277원 |

### 시간대별 매출
| 시간대 | 매출 | 비율 |
|--------|------|------|
| 18:00~20:00 | 5,200,000원 | 28.2% 🔥 |
| 20:00~22:00 | 6,800,000원 | 36.9% 🔥🔥 |
| 22:00~24:00 | 4,100,000원 | 22.2% |
| 12:00~14:00 | 950,000원 | 5.1% |
| 기타 | 1,400,000원 | 7.6% |

### AI 인사이트
1. **20시~22시 피크 집중**: 전체의 37% — 이 시간대 직원 배치를 1명 추가하면 회전율 개선 여지 있음
2. **점심(12~14시) 매출 부진**: 5%로 저조. 런치 특별 메뉴 또는 휴점 검토 가능
3. **하이볼 시리즈 매출 1위**: 수량 기준 전체의 22%. 연계 안주 세트 구성 검토 추천

## 활용 팁

- **파일이 클 경우**: 1주일 단위로 잘라서 업로드하세요. AI가 더 빠르게 처리합니다.
- **컬럼명이 다른 경우**: "날짜 컬럼은 '거래일', 금액 컬럼은 '결제금액'으로 되어 있어"처럼 알려주면 자동 인식합니다.
