# 프롬프트 04: 월간 대시보드 & 전략 인사이트

> 한 달치 데이터를 한눈에 볼 수 있는 대시보드를 생성합니다.
> 월말 결산, 가맹본부 보고, 직원 공유 등 다양한 용도로 활용하세요.

---

## 사전 확인 (실행 전 체크)

- [ ] 한 달치 CSV 파일이 준비되어 있다 (또는 주간 CSV 4개 합산)
- [ ] 전월 데이터가 있으면 비교 분석이 가능하다 (선택)
- [ ] 월 매출 목표가 있으면 입력해두면 달성률을 확인할 수 있다

---

## 프롬프트 본문

```
당신은 {회사명}의 월간 매출 대시보드를 생성하는 AI 에이전트입니다.

[매장 정보]
- 매장명: {회사명}
- 업종: {업종}
- 분석 월: {분석월}  <!-- 예: 2026년 4월 -->
- 전월 비교: {전월비교여부}  <!-- 있음 / 없음 -->
- 월 매출 목표: {월목표매출}  <!-- 예: 70,000,000원 / 없으면 "없음" -->

[첨부 파일]
- 이번 달 CSV: {이번달CSV경로}
- 전달 CSV: {전달CSV경로}  <!-- 없으면 "없음" -->

[대시보드 구성]
다음 항목을 포함한 월간 대시보드를 생성해주세요:

1. **월간 핵심 지표**
   - 총 매출, 목표 달성률(목표 있는 경우), 전월 대비 증감
   - 하루 평균 매출, 평균 객단가, 총 거래 건수

2. **주차별 흐름**
   - 1주~4주 매출 흐름 비교 (상승/하락 추세 파악)
   - 가장 좋은 주차 / 부진한 주차 분석

3. **요일별 패턴 (4주 평균)**
   - 각 요일의 월 평균 매출
   - "이 요일은 항상 좋다 / 항상 약하다" 패턴 확인

4. **시간대별 패턴 (4주 평균)**
   - 각 시간대 월 평균 매출
   - 이 데이터 기반 최적 인원 배치 제안

5. **메뉴 성과 분석**
   - 이달 Top 5 매출 메뉴 (금액 기준)
   - 이달 Top 5 인기 메뉴 (수량 기준)
   - 매출 하위 메뉴 (메뉴 정리 후보)
   - 전월 대비 순위 변동 (데이터 있는 경우)

6. **목표 달성 현황** (목표 있는 경우)
   - 주차별 목표 달성률 추이
   - 남은 기간 일평균 필요 매출 계산

7. **월간 전략 인사이트 5가지**
   - 이달 잘 된 것 2가지 (유지해야 할 것)
   - 이달 아쉬운 것 2가지 (개선할 것)
   - 다음 달 액션 1가지 (가장 임팩트 있는 것)

8. **다음 달 예측**
   - 이달 패턴 기반 다음 달 매출 예측 범위
   - 예측 근거 1~2가지

결과를 구글시트({시트URL})의 "월간대시보드_{분석월}" 탭으로 저장해주세요.
시작 전에 "이렇게 진행할게요"로 계획을 먼저 말해주세요.
```

## 출력 형식 예시

### 4월 월간 대시보드

**핵심 지표**
| 지표 | 이번 달 | 전달 | 변화 | 목표 대비 |
|------|--------|------|------|---------|
| 총 매출 | 74,500,000원 | 68,200,000원 | ▲ 9.2% | 106.4% ✅ |
| 하루 평균 | 2,483,333원 | 2,273,333원 | ▲ 9.2% | - |
| 평균 객단가 | 36,500원 | 34,100원 | ▲ 7.0% | - |

**주차별 흐름**
| 주차 | 매출 | 비고 |
|------|------|------|
| 1주 (3/31~4/6) | 17,200,000원 | 개강 시즌 효과 |
| 2주 (4/7~4/13) | 16,800,000원 | 평이 |
| 3주 (4/14~4/20) | 20,100,000원 | 🔥 최고 주차 |
| 4주 (4/21~4/30) | 20,400,000원 | 🔥 월말 강세 |

**월간 전략 인사이트**
1. ✅ 금·토 저녁 매출이 전체의 38%. 주말 드링크 패키지 구성이 효과적
2. ✅ 객단가 7% 상승. 최근 도입한 프리미엄 안주 세트 기여
3. ⚠️ 화·수요일 평균 1,700,000원으로 저조. 미드위크 이벤트 필요
4. ⚠️ 낮 12~17시 매출 비중 6%로 비효율. 점심 운영 축소 또는 폐점 검토
5. 📌 5월 액션: 어버이날(5/8) 커플 패키지 사전 예약 프로모션 시작

**다음 달 예측**
- 예측 매출 범위: 72,000,000 ~ 80,000,000원
- 근거: 5월 가정의 달 시즌 수요 + 이달 객단가 상승 추세 유지

## 활용 팁

- **가맹본부 보고용**: "대시보드를 PPT용 요약 형식으로 다시 만들어줘"를 추가하면 발표 자료 형식으로 변환됩니다.
- **직원 공유용**: "핵심 내용만 간단하게 1페이지로 요약해줘"를 추가하면 직원 공유용 버전이 나옵니다.
- **가격 정책 검토**: "메뉴별 마진율을 계산하고 싶어"를 추가하면 원가 데이터 입력 안내를 받을 수 있습니다.
