## 2. 이 프로젝트의 목적

마케팅 업무 중 반복 발생하는 작업을 자동화하고, 광고 성과를 즉시 파악하여 소재 기획까지 연결합니다.

**자동화 범위 (4축):**
- **축 A**: 블로그 콘텐츠 자동 포스팅 (워드프레스) / 네이버는 글감 보조
- **축 B**: 인스타그램 콘텐츠 스케줄링 (비즈니스 계정)
- **축 C**: 메타 광고 성과 자동 리포트 + GA4 인사이트
- **축 D**: 크리에이티브 브리프 자동 생성 (카피 + 소재 기획)

## 3. 데이터 소스

- **Meta 광고 데이터**: Meta Ads Manager (액세스 토큰 필요)
- **GA4 데이터**: Google Analytics 4 (API 키 또는 데이터 내보내기)
- **인스타그램**: Instagram Graph API (비즈니스 계정 필수)
- **워드프레스**: WordPress REST API (사이트 URL + 앱 비밀번호)
- **광고 성과 시트**: {시트URL}

## 4. AI 응답 원칙 (반드시 지킬 것)

1. **코딩 설명 최소화**: 대표님은 개발자가 아닙니다. 코드 블록은 복붙용으로만 제공하고, 설명은 **무엇을 하는지 / 어떤 결과가 나오는지**에 집중하세요.
2. **단계별 확인**: 자동화 실행 전, 항상 "이렇게 진행할게요. 진행할까요?" 형식으로 확인을 받으세요.
3. **결과는 표로 시각화**: 성과 데이터, 비교 분석 모두 마크다운 표 또는 구글시트 형식으로 정리.
4. **에러는 해결책부터**: 에러가 나면 원인 분석보다 "이렇게 해결하면 됩니다" 먼저 제시.
5. **법적·정책 이슈 사전 안내**:
   - 네이버 블로그 자동 포스팅은 공식 API 없음. 자동화 시도 시 계정 차단·노출 페널티 가능. 글감 보조로만 사용.
   - Instagram 자동화는 반드시 비즈니스/크리에이터 계정에서만. 과도한 자동화는 Account Health Score 하락 경고.
   - Meta/GA4 API 토큰은 대화에 직접 입력하지 말고 환경변수(.env)로 관리.
6. **운영 형태 분기**: `{운영형태}`가 "대행사"이면 클라이언트별 분리 리포트 구조로 자동 전환.
7. **한국어로 응답**: 모든 답변은 한국어로.

## 5. 사용할 도구

- **Antigravity** (또는 Claude Code) — 이 프로젝트의 AI 에이전트
- **Meta Ads Manager** — 광고 성과 데이터 소스
- **Google Analytics 4** — 웹 트래픽 및 전환 데이터
- **Instagram Graph API** — SNS 자동 예약 발행
- **WordPress REST API** — 블로그 자동 포스팅
- **구글 스프레드시트** — 성과 리포트 저장 및 공유
