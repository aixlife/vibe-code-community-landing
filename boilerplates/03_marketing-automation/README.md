# 카테고리 3: 마케팅 자동화 보일러플레이트

**대상**: 자사 브랜드 운영자 / 마케팅 대행사 대표님  
**핵심 가치**: 반복되는 콘텐츠 발행·광고 리포트 업무를 자동화하여 기획에 집중  
**수준**: 코딩 지식 불필요 (Antigravity + Claude로 진행)

---

## 이 보일러플레이트로 만들 수 있는 것

| 축 | 결과물 | 소요 시간 |
|----|--------|----------|
| A (블로그) | 워드프레스 자동 포스팅 파이프라인 | 세션 내 1~2시간 |
| B (SNS) | 인스타그램 주간 콘텐츠 예약 발행 | 세션 내 1~2시간 |
| C (광고) | 메타 광고 + GA4 자동 성과 리포트 | 세션 내 1.5~2시간 |
| D (소재) | 크리에이티브 브리프 & 카피 자동 생성 | 세션 내 1시간 |
| 통합 | 월간 콘텐츠 캘린더 자동 생성 | 세션 내 30분 |

---

## 사전 준비 사항

### 필수
- [ ] Antigravity 설치 및 Claude Pro 연결
- [ ] Meta 광고 계정 액세스 토큰 발급 (Meta for Developers)
- [ ] 인스타그램 비즈니스 계정 전환 (개인 계정 불가)

### 선택 (해당 축 사용 시)
- [ ] 워드프레스 사이트 + 앱 비밀번호 발급 (축 A)
- [ ] GA4 속성 ID + API 활성화 (축 C-보조)

### 주의사항 (반드시 읽으세요)
- **네이버 블로그**: 공식 자동 포스팅 API 없음. 자동화 시도 시 계정 차단·노출 페널티 가능. 이 보일러플레이트는 글감 준비 보조만 지원.
- **인스타그램**: 비즈니스/크리에이터 계정 필수. 개인 계정은 Graph API 접근 불가.
- **API 토큰**: 대화창에 직접 입력 금지. 반드시 `.env` 파일로 관리.

```

├── CLAUDE.md          ← Antigravity가 첫 번째로 읽는 컨텍스트

├── variables.md       ← 신청서 → 변수 매핑 가이드
└── prompts/
    ├── 01-wordpress-autopublish.md   ← 축 A: 워드프레스 자동 포스팅
    ├── 02-instagram-scheduler.md     ← 축 B: 인스타그램 스케줄링
    ├── 03-meta-ads-report.md         ← 축 C: 메타 광고 성과 리포트
    ├── 04-ga4-insights.md            ← 축 C 보조: GA4 인사이트
    ├── 05-creative-brief.md          ← 축 D: 크리에이티브 브리프
    └── 06-content-calendar.md        ← 통합: 월간 콘텐츠 캘린더
```

---

## 시작하는 방법

1. `CLAUDE.md`를 열고 `{}` 변수를 모두 채운다
2. `variables.md`를 참고해 신청서 답변을 변수에 매핑한다
3. Antigravity에서 이 폴더를 열고 원하는 프롬프트를 붙여넣는다
4. 자동화 순서: `CLAUDE.md` 로딩 → 원하는 축의 프롬프트 실행
