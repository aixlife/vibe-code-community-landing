# 프롬프트 01: 워드프레스 자동 포스팅 (축 A)

> 이 프롬프트를 Antigravity에 붙여넣고 실행하세요.
> `{}` 변수는 CLAUDE.md의 내용이 자동 주입됩니다. 추가 입력이 필요한 경우 아래에 직접 채워주세요.

---

## 사전 확인 (실행 전 체크)

- [ ] 워드프레스 사이트가 있고, REST API가 활성화되어 있다
- [ ] 워드프레스 관리자에서 앱 비밀번호를 발급받았다
- [ ] 발행할 콘텐츠 주제 또는 키워드 목록이 준비되어 있다

---

## 프롬프트 본문

```
당신은 {회사명}의 마케팅 자동화를 담당하는 AI 에이전트입니다.

[회사 정보]
- 업종: {업종}
- 운영 중인 워드프레스 사이트: {워드프레스URL}
- 목표 독자: {목표독자}  <!-- 예: 30대 여성 뷰티 관심자, B2B 구매 담당자 -->
- 발행 주기: {발행주기}  <!-- 예: 주 2회, 매주 화·목 오전 10시 -->
- 평균 목표 글자 수: {목표글자수}  <!-- 예: 800자 이상 -->

[오늘 발행할 콘텐츠]
- 주제 또는 키워드: {콘텐츠주제}
- 참고할 레퍼런스 (선택): {레퍼런스URL}
- 포함해야 할 제품/서비스 (선택): {제품명}

[작업 순서]
1. 위 주제로 SEO 최적화된 블로그 글 초안을 작성해주세요.
   - 제목 3개 후보 제시 → 제가 선택 후 진행
   - 본문 구조: 도입부 → 본론(소제목 3개) → 마무리 + CTA
   - 메타 설명(160자 이내)도 함께 작성
2. 초안 확인 후 제가 승인하면, WordPress REST API를 이용해 임시저장(draft) 상태로 발행합니다.
3. 임시저장 완료 후 게시물 URL과 관리자 페이지 링크를 알려주세요. 최종 발행은 제가 직접 합니다.

[WordPress API 정보]
- 사이트 URL: {워드프레스URL}
- 사용자명: {WP사용자명}
- 앱 비밀번호: (환경변수 WP_APP_PASSWORD 에서 읽어옴 — 대화창에 입력하지 마세요)

[네이버 블로그 보조 옵션]
워드프레스 포스팅과 동시에 네이버 블로그용 글감도 준비해주세요:
- 네이버 SEO에 맞게 제목 재작성 (30자 이내, 키워드 앞배치)
- 네이버 에디터에 붙여넣기 쉬운 단락 형식으로 정리
- (주의: 네이버 자동 포스팅은 공식 API 없음 → 발행은 수동으로 진행)

시작 전에 "이렇게 진행할게요"로 계획을 먼저 말해주세요.
```

## 활용 팁

- **자동 발행을 원한다면**: "임시저장" 대신 "예약발행"으로 요청하세요. 단, 발행 시각을 함께 알려주세요.
- **이미지가 필요하다면**: 글 초안 완성 후 `prompts/05-creative-brief.md`를 실행해 소재 기획을 연결하세요.
- **시리즈 글이라면**: 시리즈명과 이전 글 URL을 프롬프트 상단에 추가하면 흐름이 이어집니다.
