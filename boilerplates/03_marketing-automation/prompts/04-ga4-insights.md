# 프롬프트 04: GA4 웹 인사이트 (축 C 보조)

> 메타 광고 성과(03)와 연결하면 "광고 클릭 → 웹사이트 전환" 풀 퍼널을 확인할 수 있습니다.
> GA4 API 연동이 복잡하면 "CSV 내보내기 분석 모드"로 시작하세요 (하단 참고).

## 사전 확인 (실행 전 체크)

**[API 모드]**
- [ ] GA4 속성 ID(G-XXXXXXXX)를 알고 있다
- [ ] Google Cloud Console에서 GA4 Data API를 활성화했다
- [ ] 서비스 계정 JSON 키 파일을 발급받았다

- [ ] CSV 파일 경로를 알고 있다

## CSV 내보내기 방법 (API 설정이 어려운 경우)

1. GA4 접속 → 왼쪽 메뉴 "보고서" → "획득" → "트래픽 획득"
2. 상단 날짜 범위 설정
3. 우측 상단 "공유" 아이콘 → "파일 다운로드" → CSV 선택
4. 다운로드된 CSV 파일을 Antigravity 채팅창에 첨부

> API 대신 CSV 모드로 시작해도 동일한 인사이트를 얻을 수 있습니다.
