# PRD: 대표 AI 코칭 사전 신청 자동화 시스템

## 개요

소규모 대표 대상 2주 AI 실전 코칭 프로그램의 신청 접수부터 참가 확정까지를 자동화하는 시스템.
신청서 수집 → AI 자동 유형 판단 → 유형별 응답 초안 자동 작성 → 민수 검토 → 이메일 + 문자 발송.

---

## 목표

1. 신청서 접수 즉시 AI가 기준대로 유형 판단 및 초안 작성
2. 민수는 초안 검토 후 발송 버튼만 누르는 구조
3. 신청서 자체가 "AI에게 구체적으로 말하는 법"을 미리 체험하는 경험이 되도록
4. 통과자에게 이메일 + 문자 초안 동시 생성

---

## 사용자

- **민수 (관리자)**: 초안 검토, 수정, 발송 확정
- **신청자 (대표)**: 신청서 작성, 피드백 수신

---

## 신청서 항목

총 10개. Google Form 또는 Tally.

| 번호 | 질문 | 형식 |
|------|------|------|
| 1 | 성함 | 단답 (필수) |
| 2 | 연락처 (휴대폰 번호) | 단답 (필수) |
| 3 | 이메일 주소 | 단답 (필수) |
| 4 | 회사명 / 업종 | 단답 (필수) |

### 진단 질문 (핵심)

| 번호 | 질문 | 형식 |
|------|------|------|
| 5 | 지금 이걸 손으로 어떻게 하고 계세요? "매일 아침 나라장터 들어가서 키워드 검색하고 엑셀에 복사해요"처럼 순서대로 적어주세요. | 장문 (필수) |

| 8 | 본인만 쓰나요, 직원들도 써야 하나요? | 선택 (나만 사용 / 직원도 사용) |

### 필터 항목

| 번호 | 질문 | 형식 |
|------|------|------|
| 9 | Claude Pro 또는 ChatGPT Plus 유료 구독 중이신가요? | 선택 (예 / 아니오) |

**폼 하단 안내 문구**

> 접수 후 1~2일 내에 나민수 대표가 직접 검토하고 연락드립니다.

## 자동화 흐름

```
신청서 제출
    ↓
Google Sheets에 자동 적재
    ↓
Apps Script 트리거 감지
    ↓
Gemini API 호출 → 유형 판단 + 이메일 초안 + 문자 초안(A만) 작성
    ↓
Sheets 대시보드에 초안 자동 기입
    ↓
민수가 초안 검토 + 필요시 직접 수정
    ↓
발송 버튼 클릭 → 이메일 발송 (Gmail API)
    ↓
A 유형: 문자 초안 확인 후 발송 버튼 클릭 → 문자 발송 (CoolSMS)
```

## 유형 판단 기준

신청서 5~10번 답변을 분석해서 아래 기준으로 분류.

| 유형 | 판단 기준 |
|------|-----------|
| A — 참가 확정 | 5번이 구체적 (동사+순서 포함), 7번 출력 형태 명확, 9번 구독 있음 |
| B — 보완 요청 | 방향은 있지만 5번이 막연하거나 7번 출력 형태 없음. 질문 2개로 보완 가능한 수준 |
| C1 — 다음 기수 (막연) | 5번이 "AI로 업무 자동화하고 싶어요" 수준. 구체적 업무 흐름 없음 |
| C2 — 다음 기수 (구독 없음) | 9번 아니오 |
| C3 — 다음 기수 (범위 큼) | 5번 과업이 2주 완성 불가능한 크기 (전사 시스템, ERP 연동 등) |

## Gemini API 프롬프트

Apps Script에서 아래 구조로 Gemini API 호출.

**엔드포인트**: `https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent`  
**인증**: Google AI Studio에서 발급한 API Key

```
[시스템 프롬프트]
너는 나민수 대표의 AI 코칭 프로그램 신청서를 검토하는 어시스턴트야.
아래 신청서 답변을 보고 유형을 판단하고, 해당 유형의 응답 이메일 초안을 작성해줘.
발신자는 나민수 대표이고, 수신자는 신청자 대표님이야.
말투는 정중하지만 솔직하고 친근하게.

판단 기준:
- A (참가 확정): 5번이 구체적(동사+순서 포함), 7번 출력 형태 명확, 9번 구독 있음
- B (보완 요청): 방향은 있지만 디테일 부족. 보완 가능한 수준
- C1 (다음 기수 - 막연): 구체적인 업무 흐름 없음
- C2 (다음 기수 - 구독 없음): 유료 구독 없음
- C3 (다음 기수 - 범위 큼): 2주 완성 불가능한 크기

출력 형식 (JSON):
{
  "유형": "A/B/C1/C2/C3",
  "판단이유": "한 줄",
  "과업요약": "신청자 과업을 한 줄로",
  "이메일제목": "",
  "이메일본문": "",
  "문자초안": "A 유형만, 80자 이내. 나머지는 빈 문자열"
}

[유저 프롬프트]
신청자 이름: {이름}
5번 (업무 과정): {답변}
6번 (빈도와 시간): {답변}
7번 (원하는 결과물): {답변}
8번 (사용 범위): {답변}
9번 (구독 여부): {답변}
```

## 유형별 이메일 템플릿

Gemini가 아래를 참고해 신청자별로 개인화된 초안 생성.

### A — 참가 확정
```
제목: [AI 코칭 1기] 참가 확정 안내드립니다

안녕하세요 {이름} 대표님, 나민수 대표입니다.

신청서 내용 확인했습니다.
{과업 한 줄 요약}은 2주 안에 충분히 만들 수 있는 방향이 보입니다.

곧 과업에 맞는 사전 준비 가이드를 따로 보내드리겠습니다.
일정과 장소는 가이드와 함께 안내드립니다.

감사합니다.
나민수 드림
AIXLIFE | 010-XXXX-XXXX
```

**문자 초안 (A 유형만, 80자 이내)**
```
[AIXLIFE] {이름} 대표님, AI코칭 1기 참가 확정되었습니다.
준비 가이드 곧 이메일로 보내드립니다. -나민수
```

### B — 보완 요청
```
제목: [AI 코칭 1기] 신청서 확인 — 한 가지만 더 여쭤볼게요

안녕하세요 {이름} 대표님, 나민수 대표입니다.

신청서를 보니 {파악된 방향}은 명확합니다.

한 가지 더 확인이 필요합니다.
AI는 내가 구체적으로 말해준 만큼만 움직입니다.
아래만 추가해주시면 바로 참가 확정 드리겠습니다.

{Gemini가 생성한 맞춤 보완 질문 1~2개}

회신 기다리겠습니다.
나민수 드림
AIXLIFE | 010-XXXX-XXXX
```

### C1 — 다음 기수 (막연)
```
제목: [AI 코칭 1기] 신청서 확인 — 솔직하게 말씀드립니다

안녕하세요 {이름} 대표님, 나민수 대표입니다.

신청서 내용 잘 받았습니다.

솔직하게 말씀드리겠습니다.
지금 작성하신 내용으로는 2주 안에 무엇을 만들어드릴 수 있을지 파악이 어렵습니다.

이게 단점이 아닙니다.
AI도 같은 상황입니다. 내가 구체적으로 말해줄수록 정확하게 만들어줍니다.

참가 전에 이것 하나만 해보세요.
매일 반복하는 일 하나를 골라서, 그 일을 어떻게 하는지 순서대로 적어보세요.
"나라장터 들어가서 → 키워드 치고 → 엑셀에 복사하고" 이런 식으로요.

준비되시면 같은 링크로 다시 신청해주세요.
다음 기수에 꼭 함께하겠습니다.

나민수 드림
AIXLIFE | 010-XXXX-XXXX
```

### C2 — 다음 기수 (구독 없음)
```
제목: [AI 코칭 1기] 신청서 확인 — 사전 준비 안내

안녕하세요 {이름} 대표님, 나민수 대표입니다.

신청서 내용 확인했습니다.

한 가지 말씀드리면, 이 프로그램은 Claude Pro 또는 ChatGPT Plus 유료 구독이 반드시 필요합니다.
무료 계정은 하루 사용량 제한으로 세션 중 도구가 멈추는 상황이 생깁니다.

월 3만원 내외입니다.
구독 후 같은 링크로 재신청해주시면 바로 검토하겠습니다.

다음 기수에 뵙겠습니다.
나민수 드림
AIXLIFE | 010-XXXX-XXXX
```

### C3 — 다음 기수 (범위 큼)
```
제목: [AI 코칭 1기] 신청서 확인 — 범위를 조금 좁혀보면 어떨까요

안녕하세요 {이름} 대표님, 나민수 대표입니다.

신청서 보니 {파악된 과업}을 생각하고 계신 것 같습니다.

솔직하게 말씀드리면, 이건 2주 안에 만들 수 있는 크기가 아닙니다.
큰 걸 한번에 만들려고 하면 대부분 아무것도 완성이 안 됩니다.

제가 권장하는 방법은 이렇습니다.
그 큰 그림에서 제일 자주, 제일 오래 걸리는 일 하나만 뽑아서 그것부터 만드는 겁니다.
그게 완성되면 다음 걸 붙입니다.

{Gemini가 답변 기반으로 제안한 가장 적합한 과업 하나}만 잘라서
신청해주시면 바로 검토하겠습니다.

나민수 드림
```

## 구현 순서

```
1단계: Google Form 10개 항목 구성 + Sheets 연동 확인
2단계: Apps Script — 신규 행 감지 트리거 설정
3단계: Gemini API 연동 (gemini-2.5-flash) — 유형 판단 + JSON 초안 수신 + Sheets 자동 기입
4단계: Gmail API 연동 — 이메일 발송 버튼 기능 구현
5단계: CoolSMS API 연동 — 문자 발송 버튼 기능 구현 (A 유형만)
6단계: 마감일 기준 자동 응답 트리거 추가
7단계: 전체 테스트 — 유형별 1건씩 테스트 신청 후 흐름 확인
```
